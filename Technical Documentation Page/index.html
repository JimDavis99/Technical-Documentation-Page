Introduction

This will walk you through the fundamentals of CSS and how it works.

About CSS

CSS (Cascading Style Sheets) allows you to create great-looking web pages, but how does it work under the hood? This article explains what CSS is with a simple syntax example and also covers some key terms about the language.

Examples
This code changes how a navigation bar is displayed on a web page: nav { text-align: center; margin: 0; background-color: #5d5c7a; width: auto }

This code changes how list elements are displayed on a web page:

li { display: inline; width: auto }

This code changes how the code elements used in this examples section are displayed:

code { display: block; text-align: left; white-space: pre-line; position: relative; word-break: normal; word-wrap: normal; line-height: 2; background-color: #f7f7f7; padding: 15px; margin: 10px; border-radius: 5px; }

This code changes how links transform when the cursor is hovering on them:

a:hover { background-color: #5c687a; color: white }

This code changes how the entire body element on this webpage is displayed:

body { min-width: 290px; color: #4d4e53; background-color: #c3d2e8; font-family: 'Open Sans', Arial, sans-serif; line-height: 1.5; text-align: center }
Selectors

CSS selectors are what you use to target and style the HTML elements in a HTML document, they would be targeted in a CSS document as shown in theExamples section.

Selectors can vary in types for example you have simple selectors, which consist of an

  • id selector (#id-selector)
  • a class selector(.class-selector)
  • a universal selector(*)
  • a standdard element selector(element)
  • multi-element selector(element, element)
